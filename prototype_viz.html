JSA Export Dashboard — Visualization Prototypes

Sample data: Corn exporters · CY 2024/25 YTD through March · USDA Oct–Sep MY

OPTION 1 Quadrant Scatter — "Where's the Outlier?"

Each country is a bubble. X-axis = % vs Olympic Avg · Y-axis = % vs Last Year · Bubble size = MYTD volume (bigger = more trade, higher importance). Four quadrants tell the story instantly — bottom-left is your red flag zone. Mixed-signal countries (one metric positive, one negative) land in the off-diagonal quadrants.
Best for: instantly spotting true outliers vs mixed signals, weighted by trade importance.

OPTION 2 Traffic Light Scorecard Grid

Rows = countries · Columns = metrics (Monthly vs LY, Monthly vs Avg, MYTD vs LY, MYTD vs Avg). Each cell is color-coded green / yellow / red with the % printed inside. Thresholds: >+5% = green · −5% to +5% = yellow · <−5% = red. Countries that are red across all four columns are your priority watches.
Best for: scanning multiple metrics across all countries at once — no digging required.

OPTION 3 Lollipop Dot Plot — Cleaner Bar Chart

Same ranking as the current bar chart but rendered as thin lines + dots — less visual noise, both metrics shown per country simultaneously. Circle = % vs Avg · Diamond = % vs LY. Countries sorted highest to lowest by % vs Avg.
Best for: a tidier version of what's already built — same info, cleaner look with 10+ countries.

OPTION 4 Running Pace Chart — "Are We On Track?"

Cumulative MYTD shipments as a % of the full-year Olympic average pace, plotted month by month. The 100% dashed line = perfectly on pace. Countries above are running hot; below = falling off. Unlike the snapshot charts, this shows when a deviation started and whether it's widening or recovering.
Best for: understanding trajectory over the year, not just where things stand today.
